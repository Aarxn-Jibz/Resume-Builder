# Resume Builder Crew

## Overview

Resume Builder Crew is to assist users in creating the perfect resume from raw text inputs. This modular system can be used to analyze, format, design, and optimize resumes for the highest compatibility with modern Application Tracking Systems (ATS).

4. **Get Your Resume:**
