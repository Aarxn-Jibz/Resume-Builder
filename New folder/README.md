# Resume Builder Crew (NodeJS & React)

## Features

- NodeJS + Express backend with modular "bots"
- React front-end for user interaction
- Resume parsing, ATS scoring, and exporting

## Quick Start

### Backend

```
cd backend
npm install
node app.js
```

### Frontend

```
cd frontend
npm install
npm start
```

## Customization

- Add more intelligent parsing using NLP libraries for skill extraction.
- Replace export with PDF/DOCX using e.g. [pdfkit](https://github.com/foliojs/pdfkit), [docx](https://github.com/dolanmiu/docx).

## Notes

This MVP illustrates core architecture and modular bot logic. For full production use, expand on parsing, ATS scoring, and export logic.
